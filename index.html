Calculer la somme des chiffres d'un entier strictement positif

def somme_chiffres(x):
    ch = str(x)
    s = 0
    for i in range(len(ch)):
        s = s + int(ch[i])
    return s
🎯
Objectif
Additionner les chiffres d'un entier (ex : somme_chiffres(123) = 6).
💡
Idée générale
Convertir x en chaîne, puis parcourir chaque caractère et l'ajouter à S après conversion en entier.
🔑
Mots-clés
Convch(x)Convertit l'entier en chaîne — chaque caractère est alors un chiffre.
Valeur(Ch[i])Convertit le caractère chiffre en entier.
📋
Étapes
  • 1Ch ← Convch(x) → convertir en chaîne
  • 2Pour chaque caractère Ch[i] : S ← S + Valeur(Ch[i])
  • 3Retourner S
⚠️
Point d'attention
x doit être strictement positif.

def somme_diviseurs(x):
    S = 0

def factorielle(n):
    F = 1
    for i in range(2, n + 1):
        F = F * i
    return F
🎯
Objectif
Calculer le produit de tous les entiers de 1 à n. Ex : 5! = 120.
💡

🔑

PGCD(a, b : entier) : entier
Calculer le plus grand commun diviseur de deux entiers non nuls
▾
Par soustraction
Fonction PGCD(a, b : entier) : entier
Début
    Tant que a ≠ b Faire
        Si a > b Alors
            a ← a - b
        Sinon
            b ← b - a
        Fin Si
    Fin Tant que
    Retourner a
Fin
Méthode d'Euclide
Fonction PGCD_Euclide(a, b : entier) : entier
Début
    Tant que b ≠ 0 Faire
        r ← a Mod b
        a ← b
        b ← r
    Fin Tant que
    Retourner a
Fin
Déclaration des objets locaux
ObjetType
rEntier (reste)
Soustraction
def PGCD(a, b):
    while a != b:
        if a > b:
            a = a - b
        else:
            b = b - a
    return a
Euclide (optimisé)
def PGCD_Euclide(a, b):
    while b != 0:
        r = a % b
        a = b
        b = r
    return a
🎯
Objectif
Trouver le plus grand entier qui divise à la fois a et b.
💡
Idée générale
Méthode 1 : soustraire le plus petit du plus grand jusqu'à égalité. Méthode 2 (Euclide) : remplacer a par b et b par a mod b — bien plus rapide.
🔑
Mots-clés
a ≠ bCondition d'arrêt soustraction.
b ≠ 0Condition d'arrêt Euclide : quand b=0, a contient le PGCD.
r ← a Mod bReste de la division — clé de l'algorithme d'Euclide.
📋
Étapes
  • 1Euclide : tant que b≠0 → r←a mod b, a←b, b←r
  • 2Retourner a
⚠️
Point d'attention
La méthode d'Euclide est beaucoup plus rapide pour de grands nombres.

▾
Méthode simple
Fonction PPCM(a, b : entier) : entier
Début
    P ← a
    Tant que P Mod b ≠ 0 Faire

    Fin Tant que

Fin
Version optimisée
Fonction PPCM(a, b : entier) : entier
Début
    Si a > b Alors
        max ← a
        min ← b
    Sinon
        max ← b
        min ← a
    Fin Si
    P ← max
    Tant que P Mod min ≠ 0 Faire
        P ← P + max
    Fin Tant que
    Retourner P
Fin
Simple
def PPCM(a, b):
    P = a
    while P % b != 0:
        P = P + a
    return P
Optimisée
def PPCM(a, b):
    if a > b:
        Max = a
        Min = b
    else:
        Max = b
        Min = a
    P = Max
    while P % Min != 0:
        P = P + Max
    return P
🎯

Trouver le plus petit multiple commun à a et b.
💡
Idée générale
On part de a (ou du plus grand) et on avance par pas de a jusqu'à trouver un multiple de b.
🔑
Mots-clés
P Mod b ≠ 0Tant que P n'est pas un multiple de b, on continue.
P ← P + aOn avance d'un multiple de a à la fois.
max, minVersion optimisée : partir du plus grand pour moins d'itérations.
📋
Étapes

  • 2Tant que P Mod b ≠ 0 : P ← P + a
  • 3Retourner P

Début

Fin
Méthode 1
def Premier(x):
    ok = True
    d = 2
    while (d <= x // 2) and ok:
        if x % d != 0:
            d = d + 1
        else:
            ok = False
    return ok and x > 1
Méthode 2
def Premier(x):
    d = 2
    while (d <= x // 2) and (x % d != 0):
        d = d + 1
    return (d > x // 2) and (x > 1)

Décomposer un entier en produit de ses facteurs premiers
▾
Algorithme
Fonction facteurs_premiers(x : entier) : chaîne
Début
    fp ← ""
    d ← 2
    Tant que x > 1 Faire
        Si x Mod d = 0 Alors
            fp ← fp + Convch(d) + "*"
            x ← x Div d
        Sinon
            d ← d + 1
        Fin Si
    Fin Tant que
    Retourner Sous_chaine(fp, 0, Long(fp) - 1)
Fin
Déclaration des objets locaux
ObjetType
fpChaîne (résultat)
dEntier (diviseur courant)
Python
def facteurs_premiers(x):
    d = 2
    fp = ""
    while x > 1:
        if x % d == 0:
            fp = fp + str(d) + "*"
            x = x // d
        else:
            d = d + 1
    return fp[:len(fp) - 1]
🎯
Objectif
Afficher la décomposition de x en facteurs premiers. Ex : facteurs_premiers(12) → "2*2*3".
💡
Idée générale
On divise x par d=2, 3, … tant que d divise x. Si d ne divise pas, on passe au suivant.
🔑
Mots-clés
x ← x Div dOn divise x — on peut retrouver le même facteur plusieurs fois.
d += 1 dans SinonOn n'avance que si d ne divise pas.
fp[:len(fp)-1]Supprime le dernier "*" en trop.
📋
Étapes
  • 1d ← 2, fp ← ""
  • 2Tant que x > 1 : si d|x → fp += str(d)+"*", x //= d ; sinon d += 1
  • 3Retourner fp sans le dernier "*"
⚠️
Point d'attention
d ne saute pas les composés — ils ont déjà été éliminés de x par leurs facteurs premiers.

# sans utiliser str.lower()
def Minus(ch):
    chmin = ""
    for i in range(len(ch)):
        if "A" <= ch[i] <= "Z":
            chmin = chmin + chr(ord(ch[i]) + 32)
        else:
            chmin = chmin + ch[i]
    return chmin

def distincts(T, i):
    j = 0
    while (j < i) and (T[j] != T[i]):
        j = j + 1
    return j == i

def Recherche_dichotomique(T, n, x):
    d = 0
    f = n - 1
    Trouve = False
    while (d <= f) and (not Trouve):
        Milieu = (d + f) // 2
        if T[Milieu] == x:
            Trouve = True
        elif T[Milieu] < x:
            d = Milieu + 1
        else:
            f = Milieu - 1
    return Trouve
🎯
Objectif
Recherche rapide dans un tableau trié — O(log n).
💡
Idée générale
Comparer x avec le milieu : si x < milieu → chercher à gauche (f←Milieu-1) ; si x > milieu → à droite (d←Milieu+1).
🔑
Mots-clés
Milieu = (d+f) Div 2Centre de la zone de recherche.
f ← Milieu-1x est plus petit → éliminer la moitié droite.
d ← Milieu+1x est plus grand → éliminer la moitié gauche.
📋
Étapes
  • 1d←0, f←n-1
  • 2Milieu ← (d+f)//2
  • 3Comparer et réduire la zone
  • 4Retourner Trouve

Algorithmes de Tri — Sélection, Bulles, Insertion
Trier un tableau T de n entiers par ordre croissant
▾
Tri par sélection
Procédure Tri_Selection(@T : Tab, n : entier)
Début
    Pour i de 0 à n - 2 Faire
        posmin ← i
        Pour j de i + 1 à n - 1 Faire
            Si T[j] < T[posmin] Alors
                posmin ← j
            Fin Si
        Fin Pour
        Si posmin ≠ i Alors
            Aux ← T[i]
            T[i] ← T[posmin]
            T[posmin] ← Aux
        Fin Si
    Fin Pour
Fin
Tri à bulles
Procédure Tri_Bulles(@T : Tab, n : entier)
Début
    Répéter
        Permut ← Faux
        Pour i de 0 à n - 2 Faire
            Si T[i] > T[i + 1] Alors
                Aux ← T[i]
                T[i] ← T[i + 1]
                T[i + 1] ← Aux
                Permut ← Vrai
            Fin Si
        Fin Pour
    Jusqu'à Non(Permut)
Fin
Tri par insertion
Procédure Tri_Insertion(@T : Tab, n : entier)
Début
    Pour i de 1 à n - 1 Faire
        aux ← T[i]
        j ← i
        Tant que (j > 0) Et (T[j - 1] > aux) Faire
            T[j] ← T[j - 1]
            j ← j - 1
        Fin Tant que
        T[j] ← aux
    Fin Pour
Fin
Tri sélection
def Tri_Selection(T, n):
    for i in range(n - 1):
        posmin = i
        for j in range(i + 1, n):
            if T[j] < T[posmin]:
                posmin = j
        if posmin != i:
            aux = T[i]
            T[i] = T[posmin]
            T[posmin] = aux
Tri bulles
def tri_bulles(T, n):
    permut = True
    while permut:
        permut = False
        for i in range(n - 1):
            if T[i] > T[i + 1]:
                aux = T[i]
                T[i] = T[i + 1]
                T[i + 1] = aux
                permut = True
Tri insertion
def Tri_Insertion(T, n):
    for i in range(1, n):
        aux = T[i]
        j = i
        while j > 0 and T[j - 1] > aux:
            T[j] = T[j - 1]
            j = j - 1
        T[j] = aux
🎯
Objectif
Trier par ordre croissant avec 3 algorithmes classiques.
💡
Idée générale
Sélection : trouver le min et le placer. Bulles : faire remonter les grands. Insertion : insérer à sa bonne place.
🔑
Mots-clés
posmin ≠ iOptimisation : n'échanger que si nécessaire.
Permut ← FauxBulles : si aucune permutation → déjà trié → arrêter.
T[j] ← T[j-1]Insertion : décaler pour faire de la place.
📋
Étapes
  • 1Sélection : trouver posmin, échanger T[i] et T[posmin]
  • 2Bulles : répéter jusqu'à aucune permutation
  • 3Insertion : pour i=1..n-1 → insérer T[i] dans [0..i-1]
⚠️
Point d'attention
Ces 3 tris sont O(n²).
